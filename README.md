# 📊 Sales Data Analysis Project

## 🚀 Overview
This project analyzes sales data using Python and SQL to extract business insights.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- MySQL

## 📈 Key Analysis
- Top selling products
- Region-wise sales
- Loss-making sub-categories
- Profit margin analysis

## 📊 Insights
- West region has highest sales
- Some sub-categories are generating loss
- Discount strategy needs optimization

## 📁 Files
- superstore_data.ipynb

## 💡 Conclusion
This project helps understand business performance and improve decision making.
